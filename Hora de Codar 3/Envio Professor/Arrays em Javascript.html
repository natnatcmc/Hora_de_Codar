Arrays (vetores) em Javascript

Arrays são estruturas de dados em JavaScript que permitem armazenar e acessar vários valores em uma única variável. Cada valor em um array é identificado por um índice, que começa em 0 para o primeiro elemento e aumenta em 1 para cada elemento subsequente. Os elementos de um array podem ser de qualquer tipo de dados, como números, strings, objetos e até outros arrays.

Adicionando e Removendo Itens de um Array

É possível adicionar ou remover itens de um array usando métodos específicos. Abaixo estão os métodos mais comuns para adicionar ou remover itens de um array:

  • push(): adiciona um ou mais elementos ao final do array
  • pop(): remove o último elemento do array e retorna o valor removido
  • unshift(): adiciona um ou mais elementos ao início do array
  • shift(): remove o primeiro elemento do array e retorna o valor removido
